# BLMP UNIVERSE HUD
## 🇧🇷 Português
Plugin simples e leve para Minecraft (Paper/Spigot) que exibe um HUD na ActionBar com informações do jogador.
---
### 📌 Funcionalidades
- HUD automático ao entrar no servidor
- Mostra:
  - Ping do jogador
  - FPS simulado (120 a 600)
- Sistema de ativar/desativar HUD
- Tema verde e preto (BLMP UNIVERSE)
---
### 🎮 Exemplo no jogo
BLMP UNIVERSE | Ping: 32ms | FPS: 487
---
### ⚙ Comando

/blmp true

Ativa o HUD

/blmp false

Desativa o HUD
---
### 🚀 Instalação
1. Baixe o arquivo `.jar`
2. Coloque na pasta `plugins`
3. Inicie o servidor
4. Pronto
---
### 🧩 Requisitos
- Paper ou Spigot
- Java 17+
- Minecraft 1.20+
---
### 📁 Observação
O FPS exibido não é real, ele é simulado pelo servidor.

## 🇺🇸 English
Simple and lightweight Minecraft plugin (Paper/Spigot) that shows a HUD in the ActionBar with player information.
---
### 📌 Features
- Automatic HUD when joining the server
- Shows:
  - Player ping
  - Fake FPS (120 to 600)
- HUD toggle system (enable/disable)
- Green and black theme (BLMP UNIVERSE)
---
### 🎮 In-game example
BLMP UNIVERSE | Ping: 32ms | FPS: 487
---
### ⚙ Command

/blmp true

Enables the HUD

/blmp false

Disables the HUD
---
### 🚀 Installation
1. Download the `.jar` file
2. Place it in the `plugins` folder
3. Start the server
4. Done
---
### 🧩 Requirements
- Paper or Spigot
- Java 17+
- Minecraft 1.20+
---
### 📁 Note
The FPS shown is not real, it is simulated by the server.
